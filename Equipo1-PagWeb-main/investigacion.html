i. ¿Qué es?

Un navegador web es un programa que nos permite navegar por internet y ver el contenido de páginas web traduciendo el código HTML y mostrándonos la información por pantalla, podemos acceder a través de ellos simplemente conociendo la url, sin ellos no se podría navegar por internet.

Algunos de los navegadores web más conocidos son:

  • Opera
  • Google Chrome
  • Safari
  • Mozilla


ii. Características principales

  • Opera:
    1. Opera está disponible para todas las plataformas.
    2. Este navegador es compatible con las extensiones de google Chrome.
    3. Es un navegador rápido, muy sencillo y con un alto rendimiento
    4. Tiene funcionalidades que nos permite bloquear publicidad y un vpn integrado que nos permite acceder de forma anónima a sitios bloqueados por nuestra geolocalización.
  • Google Chrome
    1. Chrome posee versiones para Windows, Linux, macOS, iOS y Android.
    2. Chrome posee un motor de renderizado de Javascript V8 y la función recarga de DNS.
    3. Es el único navegador que implementa esta funcionalidad, que resuelve la relación IP/Nombre de dominio y la mantiene en su cache cierto tiempo por si es solicitada nuevamente.
    4. sincronización de los marcadores, extensiones, historial de navegación, contraseñas y otros datos del usuario con los servidores de Google.
    5. Este navegador también posee la función modo incognito en la que no guarda historial ni cookies.
  • Safari
    1. Este navegador esta solo disponible para dispositivos iOS y MacOs.
    2. Está basado en el motor de renderizado webkit.
    3. Tiene un alto rendimiento y velocidad y respeta los estándares de la web.
  • Mozilla
    1. Mozilla es un navegador de código abierto completamente configurable.
    2. Tiene un alto nivel de seguridad y un alto rendimiento.

iii. ¿Cuál es su origen?

El primer navegador lo desarrolló Tim-Berners-Lee en 1990 y lo llamó WorldWideWeb, pero no funcionaba en todos los ordenadores. Mas tarde salió Mosaic que empezó a funcionar en Unix y poco mas tarde crearon versiones para Windows y mac.

En 1995 lanzaron Opera desarrolladores noruegos.

Microsoft mas tarde se hizo con el mercado con el lanzamiento de Internet Explorer y al liberar su código fuente nació Mozilla el cual usaba como base el conjunto de widgets.

Apple no se queda atrás y lanza Safari en 2003 con lo que se hizo con casi la totalidad de las computadoras mac.

En 2008 google enseñó al mundo Google Chrome, este navegador en 3 años consiguió ser el navegador mas utilizado a nivel mundial.

v. Arquitectura y componentes de la arquitectura.

Los componentes de esta arquitectura son:

  • Interfaz de Usuario: Interfaz entre el usuario y el motor de buscador.
  • Motor del buscador o navegación: Componente que ofrece una interfaz de alto nivel para el motor de renderizado.
  • Subsistema de renderizado: Es el encargado de producir representación visual del recurso obtenido a partir del acceso a una direccion web. Se encarga de interpretar el código de la página web. Muestra documentos HTML, XML, CSS e incluso imágenes, videos y audio.
  • Comunicaciones: Encargado de implementar los protocolos de transferencia de ficheros y documentos utilizados en internet (HTTP, FTP, etc.). Responsable de identificar la codificación de los datos obtenidos en función de su tipo, identifica si el recurso obtenido es texto, audio, video, etc.
  • Intérprete de JavaScript: Encargado de analizar y ejecutar el código JavaScript.
  • Parser XML: Los navegadores web suelen incluir un módulo (Parser) que permite cargar en memoria una representación en árbol (DOM) de la página.
  • Componente de Visualización: Funcionalidades relacionadas con la visualización de los contenidos de un documento HTML en una web.
  • Subsistema de persistencia de datos: Almacén de diferentes tipos de datos para los diferentes subsistemas del navegador. Relacionados con el almacenamiento de historiales de navegación y el mantenimiento de sesiones de usuario en disco.

vii. Seguridad en los navegadores web

Siempre que un navegador se comunica con una página web, dicha página, como parte de la comunicación, recolecta información acerca del navegador(con el propósito de dar formato a la página que va a ser desplegada).

Si se ha insertado un código malicioso en el contenido web, o en el peor de los casos, si esa página web ha sido especialmente designada para albergar código malicioso, las vulnerabilidades específicas de un navegador en particular pueden permitir que corra código malicioso dentro del navegador y, por ende, correrán procesos de forma no intencional (cabe recordar que un bit de información hace referencia a la identidad del navegador, permitiendo que las vulnerabilidad particulares del navegador sean explotadas).

Una vez que un atacante es capaz de correr procesos en el ordenador visitante, explotar las vulnerabilidades de seguridad puede permitir al atacante ganar acceso privilegiado(si es que el navegador no está corriendo en acceso privilegiado desde ese momento) al sistema "infectado" para poder realizar procesos y actividades aún más dañinos en la máquina o incluso en la red de la víctima.

Los navegadores web se pueden atacar de formas distintas:

  • Los sistemas operativos son atacados y un malware lee y modifica el espacio de la memoria del navegador en modo privilegiado9
  • El sistema operativo tiene un malware corriendo en un proceso secundario, dicho proceso lee y modifica el espacio en memoria del navegador en modo privilegiado.
  • Los ejecutables principales del navegador pueden ser hackeados.
  • Componentes del navegador pueden ser hackeados.
  • Los plugins del navegador pueden ser hackeados.
  • Las comunicaciones de red del navegador pueden ser interceptadas fuera del ordenador.

viii. HTML y derivados

HTML siglas en inglés de HyperText Markup Language es un lenguaje de marcado que nos permite indicar la estructura de nuestro documento mediante etiquetas. Este lenguaje nos ofrece una gran adaptabilidad, una estructuración lógica y es fácil de interpre­tar tanto por humanos como por máquinas.

A lo largo de sus diferentes versiones, se han incorporado y suprimido diversas características, con el fin de hacerlo más eficiente y facilitar el desarrollo de páginas web compatibles con distintos navegadores y plataformas (PC de escritorio, portátiles, teléfonos, inteligentes, tabletas, etc.) No obstante, para interpretar correctamente una nueva versión de HTML, los desarrolladores de navegadores web deben incorporar estos cambios y el usuario debe ser capaz de usar la nueva versión del navegador con los cambios incorporados.

El HTML se escribe en forma de «etiquetas», rodeadas por corchetes angulares (<,>,/). El HTML también puede describir, hasta un cierto punto, la apariencia de un documento, y puede incluir o hacer referencia a un tipo de programa llamado script, el cual puede afectar el comportamiento de navegadores web y otros procesadores de HTML.

HTML5 es la quinta revisión del lenguaje HTML. Posee dos variantes de sintaxis, la clásica HTML y la nueva XHTML. Aunque HTML5 ha traído bastantes novedades, uno de sus mayores beneficios es la adición de nuevas etiquetas que permiten la creación de una estructura para la web semántica.

ix. CSS

CSS siglas en inglés de Cascading Style Sheets, en español «Hojas de estilo en cascada», es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.

Es muy usado para establecer el diseño visual de los documentos web, e interfaces de usuario escritas en HTML o XHTML; el lenguaje puede ser aplicado a cualquier documento XML, incluyendo XHTML, SVG, XUL, RSS, etcétera.

Junto con HTML y JavaScript, CSS es una tecnología usada por muchos sitios web para crear páginas visualmente atractivas, interfaces de usuario para aplicaciones web y GUIs para muchas aplicaciones móviles (como Firefox OS).

La separación del formato y el contenido hace posible presentar el mismo documento marcado en diferentes estilos para diferentes métodos de renderizado, como en pantalla, en impresión, en voz (mediante un navegador de voz o un lector de pantalla), y dispositivos táctiles basados en el sistema Braille. También se puede mostrar una página web de manera diferente dependiendo del tamaño de la pantalla o tipo de dispositivo. Los lectores pueden especificar una hoja de estilos diferente, como una hoja de estilos CSS guardado en su computadora, para sobreescribir la hoja de estilos del diseñador.

La especificación CSS describe un esquema prioritario para determinar qué reglas de estilo se aplican si más de una regla coincide para un elemento en particular. Estas reglas son aplicadas con un sistema llamado de cascada, de modo que las prioridades son calculadas y asignadas a las reglas, así que los resultados son predecibles.

x. Javascript

JavaScript es el nombre de un lenguaje de programación (lenguaje computacional) para generar recursos interactivos en una página web.

JavaScript es un lenguaje imperativo basado en prototipos y orientada a objetos y que se emplea en entorno cliente .

En el ámbito de programación Web los objetos en JavaScript pueden ser creados por el desarrollador pero también existen muchos que son propios de este entorno.

El creador Brendan Eich es el creador de JavaScript, desarrolló el lenguaje con el nombre de Mocha para el navegador Netscape a mediados de la década de 1990 . Con el paso de los años JavaScript fue expandiéndose hasta ser uno de los lenguajes más utilizados a día de hoy.

JavaScript permite incorporar todo tipo de elementos dinámicos e interactivos a una página web,como un contador de visita, un reloj o una calculadora, también podemos incorporar animaciones y efectos en los textos.

xi. Applets de Java

Es un lenguaje de programación Java. Pueden ejecutarse en un navegador web uilizando la Java Virtual Machine (JVM).

Entre sus características podemos mencionar un esquema de seguridad y la comodidad de entrega de permisos.

Cuando un navegador carga una página web que contiene un applet, este se descarga en el navegador web y comienza a ejecutarse. Esto permite crear programas que cualquier usuario puede ejecutar con tan solo cargar la página web en su navegador, se conoce en términos genéricos como el contenedor de los applets

Los applets son considerados códigos de poca confianza (a excepción de algunos) ya que son compartidos por todos los usuarios de internet.

xii. Ajax

Ajax significa JavaSript asíncrono y XML (Asynchronous JavaScript and XML)

Es un conjunto de técnicas de desarrollo web que permiten que las aplicaciones web funcionen de forma asíncrona.

Es una tecnología asíncrona, en el sentido de que los datos adicionales se solicitan al servidor y se cargan en segudo plano sin interferir con la visualización ni el comportamiento de la página, aunque existe la posibilidad de configurar las peticiones como caso, no es necesario que el contenido asincrono este formateado en XML.

Ajax es una combinación de cuatro tecnologías ya existentes:

XHTML para el diseño que acompaña a la información

Document Object Model (DOM) como lenguaje de scripting por parte del usuario como JavaScript y JScript.

XMLHttpRequest para intercambiar datos de forma asíncrona con el servidor web de contenido dinámico también utulizado en el método Ajax

XML es el formato usado para la transferencia de datos solicitados al servidor, aunque cualquier formato puede funcionar, incluyendo HTML preformateado, texto plano,JSON y y hasta EBML.
